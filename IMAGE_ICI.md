# Placez votre image ici

Nommez votre image : **lozhaetledragon300dpi3.jpg**

Une fois l'image placée dans ce dossier :

```bash
git add lozhaetledragon300dpi3.jpg
git commit -m "Ajout de l'image principale du spectacle"
git push -u origin main
```

L'image sera alors accessible à l'URL déjà configurée dans le HTML :
```
https://raw.githubusercontent.com/JIMROCKBAN/lozhaprimaire/main/lozhaetledragon300dpi3.jpg
```
